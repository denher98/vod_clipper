| Main pipeline commands                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |                                                                                      |                                                                                                                                                                                                                                                                                      |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Run the full pipeline                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 | python main.py --video livestream.mp4                                                | Runs all 4 stages in order: Transcription → LLM Moment Detection → YOLO Vision Scan → Cut & Edit clips. Everything is cached so if it crashes mid-way, re-running picks up where it left off.                                                                                        |
| Skip stages (use cached results)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      | python main.py --video livestream.mp4 --skip-transcribe                              | Already transcribed, re-run from LLM detection onward                                                                                                                                                                                                                                |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | python main.py --video livestream.mp4 --skip-transcribe --skip-moments               | Already transcribed + detected moments, re-run from vision scan onward                                                                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | python main.py --video livestream.mp4 --skip-transcribe --skip-moments --skip-vision | Skip all 3 heavy stages, only redo the cut + edit                                                                                                                                                                                                                                    |
| Control which clips get made                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          | python main.py --video livestream.mp4 --max-clips 10                                 | Only process the top 10 highest-scored clips (good for testing)                                                                                                                                                                                                                      |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | python main.py --video livestream.mp4 --min-score 8.0                                | Only include moments with LLM score 8 or higher (stricter)                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | python main.py --video livestream.mp4 --max-clips 5 --min-score 7.5                  | Combine both                                                                                                                                                                                                                                                                         |
| Cut only (no editing, much faster)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | python main.py --video livestream.mp4 --cut-only                                     | Just cuts the raw clips from the video using FFmpeg — no subtitles, no zoom, no SFX. Good for quickly checking if the moment detection is finding the right spots before spending time on full editing.                                                                              |
| Test LM Studio connection                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             | python main.py --test-lm-studio                                                      | Sends a small test message to LM Studio and prints the response. Run this first before processing any video to confirm the local server is running and the model is loaded.                                                                                                          |
| Train your YOLO product detector                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      | python main.py --train-yolo                                                          | Trains a custom YOLOv8 model on your labeled product dataset in dataset/. Run this once after labeling your images in Roboflow. Takes about 1–2 hours. After it finishes, update YOLO_WEIGHTS in config.py to point to the output weights file.                                      |
| Preview word corrections                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              | python main.py --video livestream.mp4 --preview-corrections                          | Requires a cached transcript from a previous run. Shows a before/after diff of every correction that WORD_CORRECTIONS in config.py would apply — e.g. ""peroya"" → ""PROYA"". Use this to discover new misspellings after your first transcription, then add them to the dictionary. |
| Check before/after images                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             | python main.py --preview-ba                                                          | Lists every image found in your assets/before_after/ folder with file sizes. Quick sanity check that the right images are loaded before running the pipeline.                                                                                                                        |
| Setup SFX folders                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     | python main.py --setup-sfx                                                           | Creates all 4 SFX folders if they don't exist and prints their current status — showing how many audio files are in each one. Run this once to get the folder structure, then drop your .wav/.mp3 files in.                                                                          |
| Web UI                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                | streamlit run app.py                                                                 | Opens a local web interface at http://localhost:8501. Has the same features as the CLI but with a visual dashboard — drag-and-drop video upload, live progress log, clip browser with video preview, and download buttons. Good if you prefer not to use the terminal.               |
